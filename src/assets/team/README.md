# Team photos

Drop leadership photos here, named after the person's `imageKey` in
`src/data/site.ts`.

- **Cris Moen** → save the photo as `cris-moen.jpg` (square, ~640×640 works best).

Photos are picked up automatically by `src/data/teamImages.ts`. If a photo is
missing, the leadership card shows a clean initials avatar instead.
